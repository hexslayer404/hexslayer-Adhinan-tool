# HexSlayer Adhinan Tool

HexSlayer is a Termux-based intelligence utility tool for
phone number and IP information analysis.

## Features
- Phone number information (country, carrier, region)
- Optional IP intelligence
- Clean terminal interface
- Educational & research use

## Requirements
- Python 3
- phonenumbers
- opencage
- folium

## Installation
```bash
git clone https://github.com/hexslayer404/hexslayer-Adhinan-tool.git
cd hexslayer-Adhinan-tool
pip install -r requirements.txt
python hexslayer_final.py
